# Minishell

Minishell is a easy version bash.

## implementation
* can parse commands using [tree like recursive algorithm](/minishell_parser.pdf).
* can handle pipe(|), redirects(<, <<, >, >>), list of operatios(||, &&) 

## build
```bash
make
```
